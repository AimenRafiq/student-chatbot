# Student Query Chatbot

A Python chatbot that answers common student queries about university life.

## About
Built using Python. Responds to questions about admissions, fees, exams, timetables, library, hostel, transport, and general campus information. Saves chat history to a text file.

## Features
- Answers 20+ common student queries
- Covers admissions, fees, exams, library, hostel and campus life
- Saves full chat history to a timestamped text file
- Simple and fast keyword-based response system

## How to Run
```bash
python chatbot.py
```

## Sample Questions to Ask
- "What are the admission requirements?"
- "Tell me about the library"
- "When are the exams?"
- "How do I apply for a scholarship?"
- "What are the hostel details?"
